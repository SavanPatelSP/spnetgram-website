# SP NET GRAM Website

Official website for SP NET GRAM.

SP NET GRAM is an independent third-party Telegram client and ecosystem project focused on enhanced user experiences, premium memberships, coins, gems, organizations, platform services, and future ecosystem expansion.

## Features

- Homepage
- Platform Overview
- Beta Program
- Development Portal
- Public Status Center
- Roadmap
- Newsroom
- Documentation
- Careers
- Support Center
- Early Access Waitlist

## Technology Stack

- Next.js 16
- TypeScript
- React
- Tailwind CSS

## Development

Install dependencies:

npm install

Run development server:

npm run dev

Build for production:

npm run build
